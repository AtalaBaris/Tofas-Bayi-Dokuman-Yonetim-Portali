# shared/

Birden fazla feature'da kullanılan ortak bileşenler.

| Bileşen | Açıklama |
|---------|----------|
| `admin-shell` | Admin layout (sidebar + `router-outlet`) |
| `admin-sidebar` | Sol menü — tüm admin sayfalarında |
| `navbar` | (eski) üst menü iskeleti |
| `loader` | Yükleme göstergesi |
| `file-upload` | Dosya yükleme UI iskeleti |
